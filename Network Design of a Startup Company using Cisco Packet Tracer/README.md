# Network Design of a Startup using Cisco Packet Tracer

### Creative Gadgets is a creative electronic gadget selling company with few sellers, a HR and a IT guy. 
### PC1 and PC2 are sellers. PC3 is a HR & marketing person. PC0 is a IT technician.
### There are three servers DHCP Server, DNS Server and Web Server (HTTP).

## Concepts which are used:
### 1.Static and dynamic IP allocation 
### 2.DHCP 
### 3.HTTP 
### 4.DNS 
### 5.Webserver configuration 
### 6.Router Interface configuration 
### 7.Made use of ip helper-address command 
### 8.Adding extra ports to the router
